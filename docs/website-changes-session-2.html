About — Vision Frame Collision

About

On the Vision chapter, the decorative offset frame was enclosing the caption text, causing a visible collision. Reported from review and corrected.

✕ Before

The frame was positioned against the whole figure — image and caption — so its offset trapped the caption inside the frame's lower area, crossing its borders.

11

Sticky Positioning Fix

Sitewide · Technical

A long-standing CSS rule was silently disabling every sticky element on the site. Found while building the exhibition and finale, which both depend on sticky.

✕ Before

html, body { overflow-x: hidden; }

Makes body its own scroll container, which disables position: sticky for all descendants.

✓ After

html, body { overflow-x: clip; }

Identical clipping behaviour, without creating a scroll container — sticky works again.

Homepage regression-checked afterwards: the brands marquee (its own scroll container) is unaffected, and no page scrolls sideways.

Services — "What We Offer" Interaction

Services

The first version of this block used an image that chased the cursor. It rendered with visible distortion, so the interaction was replaced rather than patched.

✕ Before

A tilted image plane followed the pointer with spring lag while simultaneously scaling in — forcing constant re-rasterisation, which read as warping and blur.

✓ After

A stationary sticky viewing plate beside the index. Hovering a row crossfades the plate in place — pure opacity, no movement or rotation, so every frame renders at full quality.

A small serif caption glides between category names beneath the plate, and the active row stays lit after the cursor leaves, so there is no pop-in / pop-out. Mobile keeps inline framed images per row.

Services

Custom Cursor — Presence

Sitewide

The branded cursor was easy to lose. Two causes: the logo file carries heavy transparent padding, and the mark is near-black so it vanished on dark photography.

✕ Before

Width 72px (92px over links) — but the mark occupies only ~40% of the file, so the visible circle was ≈28px of hairline stroke.

Opacity 0.88, no contrast treatment.

✓ After

Width 96px (116px over links) — visible mark ≈38px.

Opacity 1, plus a dual halo: an ivory glow tracing the strokes for dark grounds, and a soft dark shadow for weight on light ones.

Verified on both extremes — clearly legible on the onyx "What We Offer" section where it previously disappeared, and crisp on the light Practice cards without reading as a glow.

Responsiveness Pass

Sitewide

Every change above was audited at phone (390px) and tablet (768px) widths, measuring real layout geometry rather than eyeballing.

✕ Issue found

One genuine defect: the Brands index tags overflowing the viewport on mobile (see 19).

✓ Result

Fixed and re-verified. Horizontal page overflow measured at 0px on Home, About, Services, Brands and Brand-detail — at both widths.

✓ Confirmed responsive

  • Navbar & footer — squared geometry and resized marks hold at every width.
  • About — the Philosophy plaque, all five chapters, and the finale reflow cleanly; the exhibition swaps its sticky plane for framed inline images.
  • Services — practice plaques stack, the offerings index switches to inline imagery, the process deck stacks with card edges peeking.
  • Brands — index tags stack, brand cards and detail pages clean.
Elements that extend beyond the viewport by design — the brands marquee, the About ticker, the Ken Burns backdrop scale, and the form honeypot — were confirmed as intentional and correctly clipped.
